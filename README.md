# Optitracker

Besides MDOL.dll (included in MDOL-directory), the following libraries are required:  
[Alglib](https://www.alglib.net/)  
[OpenTK](https://opentk.net/)  
[RTClientSDK](https://github.com/qualisys/RTClientSDK.Net)  
[Newtonsoft.Json](https://www.newtonsoft.com/json)  
[Mono.Zeroconf](https://www.mono-project.com/archived/monozeroconf/)  
[NatNetML](https://optitrack.com/support/downloads/developer-tools.html#natnet-sdk)
